What does JavaScript bind() do ?

Posted on June 27

Back

Let's Assume that we have a very simple JavaScript Object

var person = {
firstname: 'Strasse',
lastname : '86',
concaten: function() {
var fullname = this.firstname + ' ' + this.lastname
return fullname; } }

Now let's assume that we also have a function

var display = function(){
console.log(this.concaten) }

What will happen if we call the display which prints the concaten functionthat does belong to another object ? It will fail. That is why we use bind, in order to bind methods/functions of properties to other methods.

So instead of calling

display();

we call

display.bind(person);
